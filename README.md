## Installation
Follow these steps to run the app locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/archanarackenchath/CodingApp.git

2. Navigate to the project directory:
   ```bash
   cd CodingApp

3. Install dependencies:
   ```bash
   flutter pub get
   
4. Run the App:
    ```bash
    flutter run
